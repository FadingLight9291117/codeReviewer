# 📁 示例和演示文件夹

这个文件夹包含了智能代码审查系统的各种示例和演示脚本。

## 📚 文件说明

### 🎓 core_review_tutorial.py
**核心功能教程**
- 基础代码审查功能演示
- 最近变更审查技巧
- 自定义文件审查方法
- Markdown报告生成演示
- AI模型配置和测试

```bash
# 运行核心功能教程
cd examples
python core_review_tutorial.py
```

### 🎭 multi_prefix_showcase.py
**多前缀功能完整展示**
- 基本多前缀审查演示
- 自定义前缀配置策略
- 项目路径灵活指定
- 不同时间范围分析
- 综合代码质量评估
- 错误处理展示

```bash
# 运行多前缀功能展示
cd examples
python multi_prefix_showcase.py
```

### 🧪 system_validator.py
**系统验证器**
- 系统初始化状态检查
- AI服务连接验证
- 文件审查功能测试
- Git仓库分析能力验证

```bash
# 运行系统验证
cd examples
python system_validator.py
```

## 🚀 使用建议

1. **新用户**: 先运行 `system_validator.py` 确认系统正常工作
2. **基础功能**: 运行 `core_review_tutorial.py` 了解核心功能
3. **高级功能**: 运行 `multi_prefix_showcase.py` 学习多前缀审查

## 📖 更多信息

- 主要功能文档: `../readme.md`
- 多前缀工具: `../multi_prefix_review.py`

## 💡 注意事项

- 运行示例前请确保已配置 AI API (在 `../config.yaml` 中)
- 在 Git 仓库中运行可以体验完整功能
- 示例会生成临时文件，可以安全删除
